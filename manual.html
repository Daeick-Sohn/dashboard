2

예산계획 개별 등록

한 건씩 직접 입력할 때 사용합니다.

  • 1등록 버튼을 클릭합니다.
  • 2아래 항목을 빠짐없이 입력합니다.
  • 3내용을 확인한 후 저장 버튼을 누릅니다.

입력 항목

부서 * 업무(세부 프로그램) * 세목 * 예산액 * 담당자 연도 / 월

* 표시 항목은 필수 입력입니다.

✓ 업무 + 세부 프로그램 + 세목 조합이 동일한 항목이 이미 있으면 자동으로 덮어쓰기됩니다. 중복 등록을 걱정하지 않아도 됩니다.
3

엑셀 파일로 일괄 업로드

여러 건의 예산계획을 한 번에 올릴 때 사용합니다. 업로드 전에 엑셀 양식을 먼저 확인하세요.

  • 1엑셀 양식 다운로드 버튼으로 샘플 파일을 받습니다.
  • 2샘플 파일 형식에 맞춰 데이터를 입력합니다. 첫 번째 행(헤더)은 수정하지 마세요.
  • 3엑셀 업로드 버튼을 눌러 작성한 파일을 선택합니다.
  • 4업로드 결과 메시지에서 신규 등록 건수와 수정 건수를 확인합니다.

업로드 완료 메시지 예시

✓ 업로드 완료! 신규 5건 / 수정 2건
⚠ 업무 + 세부 프로그램 + 세목이 동일한 항목은 덮어쓰기됩니다. 수정을 원하지 않는 항목은 파일에서 제외하세요.
4

등록한 계획 수정하기

아직 제출하지 않은 계획은 언제든지 수정할 수 있습니다.

  • 1수정할 계획 카드를 클릭합니다.
  • 2수정 아이콘(연필 모양) 또는 수정 버튼을 누릅니다.
  • 3내용을 변경한 후 저장을 누릅니다.
ℹ 제출 이후 수정이 필요하면 담당 팀장에게 반려 요청 후 수정하세요.
5

계획 제출 (승인 요청)

작성이 완료된 계획을 팀장에게 승인 요청합니다. 제출 후에는 카드가 파란색 테두리로 표시됩니다.

  • 1제출할 계획 카드를 클릭합니다.
  • 2제출 버튼을 누릅니다.
  • 3확인 팝업에서 확인을 선택합니다.
⚠ 제출 후에는 직접 수정할 수 없습니다. 내용을 충분히 확인한 후 제출하세요.
6

계획 삭제 (개별 / 일괄)

개별 삭제

  • 1삭제할 계획 카드의 삭제 버튼(휴지통 아이콘)을 클릭합니다.
  • 2확인 팝업에서 확인을 누르면 삭제됩니다.

일괄 삭제

  • 1카드 목록 위의 전체 선택 체크박스를 누르거나, 각 카드의 체크박스를 개별 선택합니다.
  • 2상단에 나타나는 파란 액션 바에서 🗑 일괄 삭제 버튼을 클릭합니다.
  • 3확인 팝업에서 건수를 확인하고 확인을 누릅니다.
⚠ 삭제된 데이터는 복구되지 않습니다. 삭제 전 반드시 다시 한 번 확인하세요.

자주 묻는 질문

질문답변
엑셀 업로드 후 기존 데이터가 바뀌었어요. 같은 업무+세부 프로그램+세목 조합이면 덮어쓰기됩니다. 원래 내용이 새 파일 내용으로 교체된 것입니다.
제출한 계획을 수정하고 싶어요. 팀장에게 반려 요청을 하면 다시 수정 가능한 상태로 돌아옵니다.
승인된 계획도 삭제할 수 있나요? 권한에 따라 다를 수 있습니다. 승인된 계획 삭제가 필요하면 팀장에게 문의하세요.
체크박스가 보이지 않아요. 카드가 하나 이상 있을 때만 표시됩니다. 계획을 먼저 등록하세요.
로그인이 안 돼요. 아이디/비밀번호를 확인하거나, 담당 관리자에게 계정 초기화를 요청하세요.

Section 01

대시보드 접속 및 화면 구성

브라우저에서 안내받은 링크로 접속하면 직원 대시보드가 열립니다. 팀장 화면과 메뉴 구성이 다를 수 있습니다.

주요 화면 구성

🔝상단 헤더현재 연도/월, 계정명, 로그아웃
📋예산계획 목록내가 등록한 카드 목록, 상태 색상 구분
➕등록 버튼새 예산계획 개별 등록

Section 02

예산계획 개별 등록

한 건씩 직접 입력할 때 사용합니다.

1
등록 버튼을 클릭합니다

화면 상단 또는 목록 영역의 + 등록 버튼을 누르세요.

2
입력 항목을 채웁니다

아래 항목을 빠짐없이 입력하세요. * 항목은 필수입니다.

부서 * 업무(세부 프로그램) * 세목 * 예산액 * 담당자 연도 / 월
3
저장 버튼을 누릅니다

내용을 한 번 더 확인한 후 저장하세요.

✅자동 덮어쓰기: 업무 + 세부 프로그램 + 세목 조합이 동일하면 자동으로 덮어쓰기됩니다. 중복 등록을 걱정하지 않아도 됩니다.
📤
Section 03

엑셀 파일로 일괄 업로드

여러 건의 예산계획을 한 번에 올릴 때 사용합니다.

1
엑셀 양식을 다운로드합니다

엑셀 양식 다운로드 버튼으로 샘플 파일을 받으세요.

2
양식에 맞춰 데이터를 입력합니다

첫 번째 행(헤더)은 절대 수정하지 마세요.

⚠️동일한 조합의 항목은 덮어쓰기됩니다. 수정 원하지 않는 항목은 파일에서 제외하세요.
3
엑셀 업로드 버튼으로 파일을 선택합니다

작성한 파일을 선택하면 자동으로 업로드됩니다.

4
결과 메시지를 확인합니다
업로드 완료! 신규 5건 / 수정 2건
🔧
Section 04

등록한 계획 수정하기

아직 제출하지 않은 계획은 언제든지 수정할 수 있습니다.

1
수정할 계획 카드를 클릭합니다
2
수정 버튼(연필 아이콘)을 누릅니다
3
내용 변경 후 저장합니다
ℹ️제출 이후 수정이 필요하면 담당 팀장에게 반려 요청 후 수정하세요.
📨
Section 05

계획 제출 (승인 요청)

작성이 완료된 계획을 팀장에게 승인 요청합니다. 제출 후에는 카드가 파란색 테두리로 표시됩니다.

1
제출할 계획 카드를 클릭합니다
2
제출 버튼을 누릅니다
📨 제출취소
3
확인 팝업에서 확인을 선택합니다
⚠️제출 후에는 직접 수정할 수 없습니다. 내용을 충분히 확인한 후 제출하세요.
🗑️
Section 06

계획 삭제 (개별 / 일괄)

개별 삭제

1
삭제할 카드의 휴지통 아이콘을 클릭합니다
2
확인 팝업에서 확인을 누릅니다

일괄 삭제

1
체크박스로 카드를 선택합니다

전체 선택 체크박스 또는 각 카드 개별 선택 모두 가능합니다.

2
파란 액션 바에서 일괄 삭제를 클릭합니다
3건 선택됨
🗑 일괄 삭제선택 해제
3
건수를 확인하고 확인을 누릅니다
🚨삭제된 데이터는 복구되지 않습니다. 삭제 전 반드시 다시 한 번 확인하세요.
❓
Section 07

자주 묻는 질문

Q엑셀 업로드 후 기존 데이터가 바뀌었어요.▼
같은 업무 + 세부 프로그램 + 세목 조합이면 덮어쓰기됩니다. 원래 내용이 새 파일 내용으로 교체된 것입니다.
Q제출한 계획을 수정하고 싶어요.▼
팀장에게 반려 요청을 하면 다시 수정 가능한 상태로 돌아옵니다.
Q승인된 계획도 삭제할 수 있나요?▼
권한에 따라 다를 수 있습니다. 승인된 계획 삭제가 필요하면 팀장에게 문의하세요.
Q체크박스가 보이지 않아요.▼
카드가 하나 이상 있을 때만 표시됩니다. 계획을 먼저 등록하세요.
Q로그인이 안 돼요.▼
아이디/비밀번호를 확인하거나, 담당 관리자에게 계정 초기화를 요청하세요.

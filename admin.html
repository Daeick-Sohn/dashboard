팀원 접근 링크 관리

각 팀원에게 개인 URL을 공유하면, 로그인 없이 바로 업무 대시보드에 접근할 수 있습니다.
링크를 카카오톡, 이메일, 문자 등으로 전달하세요.

아래 각 팀원 링크를 복사해서 카카오톡/이메일로 전달하세요.
내 접근 링크 (팀장)
팀장 총괄 대시보드 전용 링크 — 반드시 안전하게 보관하세요

링크 보안 주의사항

  • 각 링크는 해당 팀원 전용입니다. 다른 사람에게 공유하지 마세요.
  • 링크를 분실하거나 유출되었을 경우, 이 페이지에서 새 링크를 발급할 수 있습니다.
  • 팀장 링크는 전체 데이터 조회·수정이 가능하므로 특히 안전하게 관리하세요.
  • 링크는 SNS, 공개 채널에 올리지 말고 1:1 메시지로만 전달하세요.

링크 재발급이 필요한 경우

팀원이 링크를 분실하거나 보안 우려가 생겼을 때는, js/common.js 파일의 토큰 값을 변경하면 즉시 기존 링크가 무효화되고 새 링크가 생성됩니다.
변경 후 이 페이지에서 새 링크를 팀원에게 재전달하세요.
